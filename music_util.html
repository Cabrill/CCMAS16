  • music_key (str) – The name of the key to be matched (i.e. ‘cmajor’)

static determine_theme(word)[source]¶

This method takes a word supplied by the agent and attempts to match items as a substring of themes encompassed by word lists. If a matching theme is found then a list of MIDI instrument numbers is returned. A substring will match (word=old, will_match=olden)

Parameters:word (str) – The word to use as a potential theme match
Returns:A tuple containing tempo and a list of MIDI instruments in int format.
