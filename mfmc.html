mfmc module¶

usage: mfmc.py [-h] [-a AGENTS] [-p PATH] [-o ORDER] [-r ROUNDS] [-m MEMORY]

optional arguments:
  -h, --help            show this help message and exit
  -a AGENTS, --agents AGENTS
                        The number of concurrent agents to simulate. DEFAULT:
                        5
  -p PATH, --path PATH  The location of the inspiring set of TXT files.
                        DEFAULT: InspiringSet
  -o ORDER, --order ORDER
                        The order of Markov Chain to use in lyric generation.
                        DEFAULT: 2
  -r ROUNDS, --rounds ROUNDS
                        The number of voting rounds to simulate. DEFAULT: 1000
  -m MEMORY, --memory MEMORY
                        The number of seen artifacts an agent can remember.
                        DEFAULT: 20

MFMC(Music From Multiple Content) project to create music(lyrics+tracks) using multiple agents in a creamas environment and using the pyknon library to create MIDI files.
